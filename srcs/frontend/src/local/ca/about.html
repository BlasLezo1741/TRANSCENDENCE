Informació del projecte

Aquesta aplicació ha estat desenvolupada com a part del projecte ft_transcendence del currículum de 42.

Objectiu

Crear una aplicació web completa que integri:
- Frontend
- Backend
- Base de dades
- Funcionalitats en temps real
- Gestió d'usuaris

Tecnologies

- Frontend: React
- Backend: (Express / Nest / Django, etc.)
- Base de dades: (PostgreSQL / SQLite / MongoDB)
- Docker per a desplegament

El projecte ha estat desenvolupat en equip, seguint bones pràctiques de desenvolupament i gestió.
