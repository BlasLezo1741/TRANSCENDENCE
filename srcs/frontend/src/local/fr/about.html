Information sur le projet

Cette application a été développée dans le cadre du projet ft_transcendence du cursus 42.

Objectif

Créer une application web complète qui intègre :
- Frontend
- Backend
- Base de données
- Fonctionnalités en temps réel
- Gestion des utilisateurs

Technologies

- Frontend : React
- Backend : (Express / Nest / Django, etc.)
- Base de données : (PostgreSQL / SQLite / MongoDB)
- Docker pour le déploiement

Le projet a été développé en équipe, en suivant les bonnes pratiques de développement et de gestion.
