Project Information

This application has been developed as part of the ft_transcendence project from the 42 curriculum.

Objective

Create a complete web application that integrates:
- Frontend
- Backend
- Database
- Real-time functionalities
- User management

Technologies

- Frontend: React
- Backend: (Express / Nest / Django, etc.)
- Database: (PostgreSQL / SQLite / MongoDB)
- Docker for deployment

The project has been developed as a team, following good development and management practices.
